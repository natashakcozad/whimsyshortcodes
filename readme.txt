== Description ==
Whimsy+Shortcodes is a lightweight plugin featuring a handful of pretty buttons, info boxes, an accordion, dividers, and more.

Don't rely on a theme to provide your site with awesome shortcodes, switch to the Whimsy+Shortcodes plugin so you can change your theme on a whim.

<h4>Features</h4>
<ol>
<li>Responsive!</li>
<li>Made with CSS3.</li>
<li>FontAwesome icons.</li>
<li>Shortcode wizard for easy use!</li>
<li>Buttons in multiple colors and sizes.</li>
<li>Custom category and recent post listing shortcodes.</li>
<li>Content boxes: Plain, Info, Download, Warning, Tick, Heart.</li>
<li>Accordion and panels for toggling content. </li>
<li>Content dividers in 3 styles.</li>
<li>Font Awesome icon shortcodes.</li>
</ol>

== Installation ==
1. Unzip and upload \"whimsy-shortcodes.zip\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Start using your pretty new shortcodes!
